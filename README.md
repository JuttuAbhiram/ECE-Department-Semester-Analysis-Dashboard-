# 🎓 ECE Semester Analysis Dashboard

> **An interactive Power BI dashboard for analyzing student semester performance using dynamic visualizations, KPIs, and filters.**

---

## 📌 Project Overview

The **ECE Semester Analysis Dashboard** is an interactive Power BI project developed to analyze academic performance of Electronics and Communication Engineering (ECE) students.

The dashboard transforms raw student data into meaningful insights, enabling users to:

* Monitor section-wise student performance
* Compare SGPA trends
* Analyze pass percentages
* Filter results by semester and other dimensions
* Make data-driven academic decisions

---

## 🚀 Dashboard Preview

> **Download the ****`.pbix`**** file from this repository and open it in Power BI Desktop to experience the full interactive dashboard.**

---

## ✨ Key Features

✅ Interactive Slicers

* Section Selection
* Department Filters

✅ KPI Cards

* Backlogs count
* Pass Percentage

✅ Interactive Visualizations

* Subject Failure Rate
* SGPA Distribution
* Top SGPA Students
* Backlog Students by Section
* Students by Backlogs

✅ Drill-down Analysis

Users can interact with every visual to discover deeper insights.
